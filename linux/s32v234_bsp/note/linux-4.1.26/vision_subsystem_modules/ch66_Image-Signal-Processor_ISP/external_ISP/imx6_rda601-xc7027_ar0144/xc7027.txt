 一组 I2C master 接口，用来控制摄像头
 I2C 旁路功能(i2c bypass)，可以通过 host 直接访问摄像头

XC7027 是一款内部集成了 32 位 RISC 处理器的高端图像信号处理（ISP） SoC 芯片， 
XC7027 支持 8/10bit RAW 信号输入， YUV422 输出， 1080P@30fps， 720@60fps。
3.2 工作过程
整个芯片工作过程步骤如下方式：
（1） 外部电源给 XC7027 上电，保持稳定供电；
（2） 供电稳定后外部上电复位信号释的 RESETB 管脚，使 XC7027 从复位状态释放出来；
（3） Host 通过 I2C slave 接口对 XC7027 进行用户自定义配置；
（4） XC7027 通过 I2C master 接口对 sensor 进行相应配置，使 sensor 进入工作状态；
（5） 开始正常工作；
