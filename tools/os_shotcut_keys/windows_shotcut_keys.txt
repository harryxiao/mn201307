win10 系统多桌面切换:
(1) 虚拟桌面
　　创建：Win + Ctrl + D
　　关闭：Win + Ctrl + F4
　　切换：Win + Ctrl +方向键左/右

(2) 窗口
　　贴靠窗口：Win + 左/右
			  Win + 上/下
			  win + 左/右 + 上/下
			  窗口可以变为1/4大小放置在屏幕4个角落
　　切换窗口：Alt + Tab(不是新的，但任务切换界面改进)
(3) 任务视图
　　任务视图：Win + Tab(松开键盘界面不会消失)
